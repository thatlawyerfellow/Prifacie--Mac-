# Third-Party Components

This project is designed around open-source components that are commonly used in commercial products.

## Engine

- Runtime-selectable engine adapter (`WebKit` default, `Gecko` stub path)
- Current full-feature macOS backend uses `WebKit` via `WKWebView`

## UI Toolkit

- `Cocoa/AppKit` (Apple frameworks)

## Crypto

- `OpenSSL` (`libcrypto`)
- License: Apache-2.0

## Icon Theme

- Font Awesome Free (Open Font License / CC BY where applicable)
- macOS SF Symbols (platform-provided fallback)

## Commercialization note

You can commercialize software built on this stack, but you must comply with the relevant open-source licenses (for example LGPL obligations for dynamically linked libraries and license notices/disclosures where required).
